Detailed Description

This class can be used to extract documentation for assemblies that are shipped with a corresponding XML documentation file, as well as types, members and methods within such assemblies.

Author: Daniel Saidi [daniel.saidi@gmail.com] Link: http://www.dotnextra.com

This class wraps the various XML extractor classes that are defined in the Extractors sub namespace. It makes it easier to work with an XML documented assembly and its components, since all functionality is gathered in one class.
