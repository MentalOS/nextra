Detailed Description

This attribute can be used to validate whether or not a string represents a valid Finnish Social Security Number.

Author: Daniel Saidi [daniel.saidi@gmail.com] Link: http://www.dotnextra.com

To validate more complex scenarios, like correct sex or region, create a separate class and override the IsValid method.
