Detailed Description

This class can translate language keys hierarchically, using a resource file. Unlike its base class, it will strip language keys, piece by piece, until it finds a translation, if one exists. /summary>

Author: Daniel Saidi [daniel.saidi@gmail.com] Link: http://www.dotnextra.com

If we take the key Domain_User_UserName, for instance, the class uses it directly, without any stripping. If no translation is found and we use _ as key separator, it then uses User_UserName, then finally UserName.
