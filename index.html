What does it contain?

Interfaces

Templates that describe useful stuff that you will probably need every once in a while.

Implementations

Implementations of useful stuff that you will probably need every once in a while.

Extensions

Extensions native classes, that make them do more than they originally do.

Utilities

Validation, documentation extraction, Windows Forms control printing etc. etc. NExtra has it!

Documentation

NExtra is thoroughly (but sensibly) documented and does not leave you out in the cold.

Test suites

NExtra is thoroughly (but sensibly) tested, to assure that working stuff do not break.

Get started

To get started, either
  • grab the source code from GitHub
  • download the pre-compiled binaries from the download page
  • grab the libraries you need directly from within Visual Studio, using NuGet

If you are new to NExtra, I would advice you to grab the source code and pick out the parts you need, instead of starting out with the compiled DLLs. Read more at the project site.

Documentation

From NExtra 3.0.0, I have decided not to extract and publish documentation for the various libraries. All classes are documented and should be easy to understand for anyone who download the source code. If you disagree, let me know.

Contribute

Contributions to NExtra are more than welcome. Just send me a pull request, attach the code in an e-mail or whatever. I'd love to have a look at it.
